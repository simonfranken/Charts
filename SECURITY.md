# Security Policy

## Reporting A Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities by email to:

- `mail@simon-franken.de`

Include as much detail as possible:

- affected chart and version,
- impact and attack scenario,
- reproduction steps,
- suggested fix (if available).

## Response Expectations

- Initial response target: within 5 business days.
- Confirmed issues will be fixed and released as soon as practical.

## Scope

This policy covers this repository's Helm charts, templates, and release process.
